# MERN Auth Template

# Project Setup In Your Local Machine
### Steps:
#### 1) Clone the project
    git clone https://github.com/prajapatiomkar/mern-auth-template
#### 2) Install the dependency
    npm run install 
#### 3) To run project
    npm run dev
#### 4) To build project 
    npm run build
